# Contributing

Welcome, and thanks for your interest in contributing to claude-got-skills!

## Issues and Feature Requests

Issues and feature requests are welcome. Please use the provided templates when opening a new issue.

## Pull Requests

PRs are welcome. Please include a clear description of what your change does and why it is needed. Keep PRs focused on a single concern where possible.

## Testing Locally

To test a plugin or skill locally, install it from your local path:

```
/plugin install /path/to/your/local/plugin
```

Verify that the plugin loads correctly and behaves as expected before submitting your PR.
